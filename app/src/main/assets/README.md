##assets资源目录
资源目录文件夹用于存放有关项目所需要用到的各种资源
###**目录结构**
```
assets/
|
|----fonts/  --------  网站字体，iconfont图标字体存放目录
|----images/ --------  图片存放目录
|----javascripts/  -------- javascipt文件存放目录
|----stylesheets/  -------- css样式文件存放目录
|    |
|    |----main.css  -------- 主样式文件，包含了bootstrap，为减少响应请求合并为一个
|
|----README.md
```
